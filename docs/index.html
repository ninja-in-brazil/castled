Castled

Simple backups for Omarchy. Opinionated defaults, zero ceremony.

Omarchy is beautiful, modern & opinionated Linux — omakase computing, chef's choice. Castled brings the same spirit to your personal dotfiles: one command to initialize, one to back up, one to restore. No backup framework to learn. Substitutions welcome.

  • Curated config.yml from simple-backup init
  • Timestamped copies to any destination (USB, NAS, local disk)
  • Interactive restore with dry-run and diff preview

Install

gem build castled.gemspec
gem install castled-*.gem

Initialize

simple-backup init
# creates config.yml with Omarchy-friendly defaults

Example config

backup_name: omarchy
backup_paths:
  - ~/.bash_logout
  - ~/.bash_profile
  - ~/.bashrc
  - ~/.gemrc
  - ~/.XCompose
destination: /run/media/leonid/250GB/backups

Backup & restore

simple-backup backup
# → omarchy_YYYYMMDD_HHMMSS/

simple-backup restore
simple-backup restore --dry-run --diff

Schedule (optional)

# crontab — daily at 9:00
0 9 * * * cd /path/to/config && simple-backup backup
